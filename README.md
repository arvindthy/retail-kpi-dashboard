# 📊 Retail KPI Dashboard (Power BI)


[![Power BI](https://img.shields.io/badge/Power--BI-Dashboard-yellow.svg)](https://powerbi.microsoft.com/)
[![Excel](https://img.shields.io/badge/Excel-Data-green.svg)](https://www.microsoft.com/en-us/microsoft-365/excel)

An interactive dashboard tracking key retail metrics like revenue, profit, and customer counts using Power BI.

## 📊 Features

- Region-wise Sales and Profit overview
- Monthly performance trends
- Dynamic filters (Region, Category, Date)

## 🧠 Skills Demonstrated
- Power BI report design
- DAX and calculated columns
- Interactive filters and visuals
- Business storytelling with data

## 📁 Files

- `Retail_Dashboard.pbix` – Power BI dashboard file
- `Retail_Sample_Data.xlsx` – Source data

## 📌 How to Use

1. Open `Retail_Dashboard.pbix` in Power BI Desktop
2. Connect it to the Excel file
3. Explore KPIs interactively
